## 🔧 **إذا واجهت مشكلة:**

### **1. تأكد من إعدادات MySQL:**
```sql
-- في phpMyAdmin
SET GLOBAL sql_mode = '';
SET SESSION sql_mode = '';
```

### **2. تأكد من صلاحيات المستخدم:**
```sql
GRANT ALL PRIVILEGES ON asha_app.* TO 'your_user'@'localhost';
FLUSH PRIVILEGES;
```

### **3. تحقق من ملف config.php:**
```php
define('DB_HOST', 'localhost');
define('DB_USER', 'root');
define('DB_PASS', '');
define('DB_NAME', 'asha_app');
```
