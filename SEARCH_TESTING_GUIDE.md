# دليل اختبار نظام البحث الجديد

## 🧪 اختبار الوظائف الأساسية

### 1. اختبار تحميل الفئات
```bash
# تأكد من أن الفئات تظهر بشكل صحيح
- افتح صفحة البحث
- يجب أن تظهر قائمة الفئات مع عدد الخدمات
- تأكد من أن الأيقونات تظهر لكل فئة
```

### 2. اختبار اختيار الفئة
```bash
# اختبار الخطوة الأولى
- اضغط على أي فئة (مثل: تصوير)
- يجب أن تنتقل للخطوة الثانية
- يجب أن تظهر الفئة المختارة في الأعلى
```

### 3. اختبار البحث عن السعر الأنسب
```bash
# اختبار البحث حسب السعر
- اختر فئة (مثل: تصوير)
- اختر "السعر الأنسب"
- أدخل الحد الأدنى: 3000
- أدخل الحد الأقصى: 5000
- اضغط "بحث عن الخدمات"
- تأكد من ظهور النتائج في النطاق المحدد
```

### 4. اختبار البحث عن الأقرب
```bash
# اختبار البحث حسب المسافة
- اختر فئة (مثل: مطاعم)
- اختر "الأقرب"
- تأكد من تفعيل GPS
- اضغط "بحث عن الأقرب"
- تأكد من ظهور النتائج مرتبة حسب المسافة
```

### 5. اختبار البحث النصي
```bash
# اختبار البحث بالكلمات
- اختر فئة
- اكتب كلمة في شريط البحث (مثل: "احترافي")
- اضغط بحث
- تأكد من ظهور النتائج المطابقة
```

## 🔧 اختبار الأخطاء والحدود

### 1. اختبار GPS غير مفعل
```bash
# اختبار البحث عن الأقرب بدون GPS
- أوقف GPS
- اختر "الأقرب"
- يجب أن تظهر رسالة: "يرجى تفعيل GPS للبحث عن الأقرب"
```

### 2. اختبار نطاق السعر الفارغ
```bash
# اختبار السعر الأنسب بدون تحديد النطاق
- اختر "السعر الأنسب"
- اترك حقول السعر فارغة
- اضغط بحث
- يجب أن تظهر رسالة: "يرجى تحديد نطاق السعر"
```

### 3. اختبار عدم اختيار فئة
```bash
# اختبار البحث بدون فئة
- انتقل للخطوة الثانية بدون اختيار فئة
- يجب ألا يتم تنفيذ البحث
```

### 4. اختبار عدم وجود نتائج
```bash
# اختبار البحث بدون نتائج
- اختر فئة نادرة
- حدد نطاق سعر ضيق جداً
- تأكد من ظهور رسالة: "لا توجد خدمات تطابق معايير البحث"
```

## 📱 اختبار واجهة المستخدم

### 1. اختبار التنقل
```bash
# اختبار زر العودة
- انتقل للخطوة الثانية
- اضغط زر العودة
- تأكد من العودة للخطوة الأولى
- تأكد من مسح البيانات
```

### 2. اختبار شريط البحث
```bash
# اختبار تفاعل شريط البحث
- اكتب نص في شريط البحث
- تأكد من ظهور زر المسح (X)
- اضغط على X لمسح النص
- تأكد من تحديث النتائج
```

### 3. اختبار الفلاتر
```bash
# اختبار الفلاتر الإضافية
- اضغط على أيقونة الفلاتر (⚙️)
- تأكد من ظهور خيارات إضافية
- اختبر تغيير نطاق السعر
- اختبر تغيير الترتيب
```

## 🌐 اختبار الاتصال بالخادم

### 1. اختبار الاتصال العادي
```bash
# اختبار مع اتصال إنترنت طبيعي
- تأكد من عمل جميع الوظائف
- تحقق من سرعة الاستجابة
```

### 2. اختبار الاتصال البطيء
```bash
# اختبار مع اتصال بطيء
- استخدم اتصال إنترنت بطيء
- تأكد من ظهور مؤشر التحميل
- تأكد من عدم تجميد التطبيق
```

### 3. اختبار عدم وجود اتصال
```bash
# اختبار بدون إنترنت
- أوقف الاتصال بالإنترنت
- حاول البحث
- تأكد من ظهور رسالة خطأ مناسبة
```

## 📊 اختبار الأداء

### 1. اختبار عدد النتائج
```bash
# اختبار مع عدد كبير من النتائج
- اختر فئة تحتوي على خدمات كثيرة
- تأكد من عمل التمرير بسلاسة
- تحقق من عدم بطء التطبيق
```

### 2. اختبار التحديث
```bash

- تأكد من تحديث النتائج فوراً
- تحقق من عدم تكرار النتائج
```

## 🚀 نصائح للاختبار

1. **اختبر على أجهزة مختلفة**: Android و iOS
2. **اختبر أحجام شاشات مختلفة**: هاتف، تابلت
3. **اختبر سرعات إنترنت مختلفة**: سريع، بطيء، بدون إنترنت
4. **اختبر سيناريوهات مختلفة**: مستخدم جديد، مستخدم متقدم
5. **اختبر البيانات المختلفة**: فئات فارغة، فئات مليئة

## 📞 الإبلاغ عن المشاكل

عند اكتشاف مشكلة:
1. **سجل الخطوات**: بالتفصيل
2. **التقط لقطة شاشة**: للمشكلة
3. **سجل رسالة الخطأ**: إن وجدت
4. **حدد الجهاز**: نوعه وإصداره
5. **أرسل التقرير**: للفريق التقني
