# دليل البحث خطوة بخطوة - النظام الجديد

## 🎯 نظرة عامة

تم تطوير نظام بحث جديد يعمل على خطوتين:
1. **الخطوة الأولى**: اختيار نوع الخدمة
2. **الخطوة الثانية**: اختيار معيار البحث (السعر الأنسب أو الأقرب)

## 🔄 خطوات البحث الجديدة

### الخطوة 1: اختيار نوع الخدمة 🏷️

#### الشاشة المعروضة:
```
┌─────────────────────────────────────────┐
│ الخطوة 1: اختر نوع الخدمة               │
│ اختر نوع الخدمة التي تبحث عنها:         │
│                                         │
│ [📷 تصوير]    [🎉 تنسيق]               │
│ [🍽️ مطاعم]    [🎵 موسيقى]              │
│ [🚗 نقل]       [👗 أزياء]               │
│ [🏠 ديكور]     [🔒 أمن]                 │
└─────────────────────────────────────────┘
```

#### الفئات المتاحة:
- **📷 تصوير**: خدمات التصوير الاحترافي
- **🎉 تنسيق**: تنسيق وتجميل المناسبات
- **🍽️ مطاعم**: خدمات الطعام والضيافة
- **🎵 موسيقى**: خدمات الموسيقى والترفيه
- **🚗 نقل**: خدمات النقل والمواصلات
- **👗 أزياء**: خدمات الأزياء والتجميل
- **🏠 ديكور**: خدمات الديكور والتجهيزات
- **🔒 أمن**: خدمات الأمن والحماية

### الخطوة 2: اختيار معيار البحث 🎯

#### الشاشة المعروضة:
```
┌─────────────────────────────────────────┐
│ الخطوة 2: اختر معيار البحث             │
│ الفئة المختارة: تصوير                   │
│                                         │
│ [💰 السعر الأنسب] [📍 الأقرب]          │
└─────────────────────────────────────────┘
```

#### الخيارات المتاحة:

##### 1. **السعر الأنسب** 💰
- **الوصف**: البحث عن الخدمات في نطاق سعر محدد
- **المميزات**:
  - تحديد الحد الأدنى والأقصى للسعر
  - ترتيب حسب السعر (الأرخص أولاً)
  - عرض نطاق سعر مخصص

##### 2. **الأقرب** 📍
- **الوصف**: البحث عن الخدمات الأقرب لموقع المستخدم
- **المتطلبات**: تفعيل GPS
- **المميزات**:
  - ترتيب حسب المسافة (الأقرب أولاً)
  - عرض المسافة لكل خدمة
  - البحث في نطاق 50 كم

## 💰 تحديد نطاق السعر

### عند اختيار "السعر الأنسب":
```
┌─────────────────────────────────────────┐
│ حدد نطاق السعر:                         │
│                                         │
│ [الحد الأدنى (ريال)] [الحد الأقصى (ريال)] │
│                                         │
│ [بحث عن الخدمات]                        │
└─────────────────────────────────────────┘
```

### مثال على الاستخدام:
- **الحد الأدنى**: 3000 ريال
- **الحد الأقصى**: 5000 ريال
- **النتيجة**: خدمات تصوير بأسعار بين 3000-5000 ريال

## 🎨 واجهة المستخدم الجديدة

### التنقل بين الخطوات:
- **زر العودة**: للعودة لاختيار الفئة
- **توجيه واضح**: عرض الخطوة الحالية
- **تحديث فوري**: للنتائج

### الألوان والتصميم:
- **الأزرق**: للخطوات والتوجيه
- **الأخضر**: للنجاح والتأكيد
- **الأحمر**: للتنبيهات والأخطاء

## 🚀 كيفية الاستخدام

### للمستخدم الجديد:
1. **افتح صفحة البحث**
2. **اختر نوع الخدمة** من القائمة
3. **اختر معيار البحث**:
   - **السعر الأنسب**: إذا كنت تبحث عن سعر محدد
   - **الأقرب**: إذا كنت تبحث عن خدمات قريبة
4. **إذا اخترت السعر الأنسب**:
   - أدخل الحد الأدنى للسعر
   - أدخل الحد الأقصى للسعر
   - اضغط "بحث عن الخدمات"
5. **تصفح النتائج**

### للمستخدم المتقدم:
- **استخدم الفلاتر الإضافية** (أيقونة ⚙️)
- **اضبط نطاق السعر** حسب ميزانيتك
- **اختر الترتيب** المناسب
- **استخدم البحث النصي** للبحث المحدد

## 📱 المميزات الجديدة

### البحث الذكي:
- **توجيه خطوة بخطوة**: لا يضيع المستخدم
- **تحقق من المتطلبات**: GPS للسعر الأنسب
- **رسائل واضحة**: توجيه المستخدم

### تجربة محسنة:
- **واجهة بديهية**: سهلة الاستخدام
- **نتائج دقيقة**: حسب المعايير المحددة
- **سرعة في البحث**: خطوات واضحة

### المرونة:
- **العودة للخطوات السابقة**: تعديل الاختيارات
- **حفظ التفضيلات**: في الجلسة الحالية
- **خيارات متعددة**: لكل نوع مستخدم

## 🎯 حالات الاستخدام

### المستخدم الاقتصادي:
1. **اختر الفئة**: مثلاً "تصوير"
2. **اختر "السعر الأنسب"**
3. **حدد النطاق**: 2000-4000 ريال
4. **تصفح الخدمات الاقتصادية**

### المستخدم المهتم بالموقع:
1. **اختر الفئة**: مثلاً "مطاعم"
2. **اختر "الأقرب"**
3. **تفعل GPS**
4. **تصفح المطاعم القريبة**

### المستخدم الفضولي:
1. **اختر الفئة**: "الكل"
2. **استخدم الفلاتر الإضافية**
3. **تصفح جميع الخدمات**

## 🔄 التحديثات المستقبلية

### ميزات مقترحة:
- **حفظ التفضيلات**: في قاعدة البيانات
- **توصيات ذكية**: بناءً على التاريخ
- **مقارنة الأسعار**: بين الخدمات المماثلة
- **إشعارات**: للخدمات الجديدة في الفئة المفضلة

### تحسينات واجهة المستخدم:
- **رسوم متحركة**: بين الخطوات
- **ألوان مخصصة**: لكل فئة
- **إحصائيات**: أكثر الفئات استخداماً
