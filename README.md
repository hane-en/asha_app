# 🎉 تطبيق خدمات المناسبات - Event Services Flutter App

## 📱 نظرة عامة

تطبيق خدمات المناسبات هو تطبيق Flutter متكامل يوفر منصة شاملة لربط العملاء بمزودي الخدمات في مجال المناسبات. يتيح التطبيق للمستخدمين البحث عن الخدمات، إضافتها للمفضلة، وحجزها بسهولة.

## ✨ الميزات الرئيسية

### 🔍 **البحث المتقدم**
- بحث في الخدمات حسب الفئة
- فلترة حسب السعر والموقع
- ترتيب النتائج حسب التقييم والسعر
- دعم GPS للبحث عن الأقرب

### ❤️ **المفضلة الذكية**
- إضافة الخدمات للمفضلة
- دعم المفضلة المحلية للمستخدمين غير المسجلين
- مزامنة المفضلة مع الخادم للمستخدمين المسجلين

### 📋 **إدارة الطلبات**
- عرض حالة الطلبات
- تتبع الطلبات المنجزة والمعلقة
- إشعارات فورية لحالة الطلبات

### 🔐 **نظام المصادقة المتطور**
- تسجيل دخول وإنشاء حساب
- تسجيل دخول للمزودين
- لوحة تحكم للمدير
- إدارة الملف الشخصي

### 🎨 **واجهة مستخدم حديثة**
- تصميم Material Design 3
- ألوان أرجوانية جذابة
- تجربة مستخدم سلسة
- دعم اللغة العربية

## 🚀 التثبيت والتشغيل

### المتطلبات الأساسية
- Flutter SDK (3.0.0 أو أحدث)
- Dart SDK (3.0.0 أو أحدث)
- PHP 8.0 أو أحدث
- MySQL 8.0 أو أحدث
- XAMPP أو WAMP أو MAMP

### خطوات التثبيت

#### 1. **إعداد الخلفية (Backend)**
```bash
# 1. نسخ ملفات PHP إلى مجلد الخادم
cp -r backend_php/ /path/to/your/web/server/

# 2. إنشاء قاعدة البيانات
mysql -u root -p < database/create_database.sql

# 3. تعديل إعدادات الاتصال
# افتح backend_php/config/database.php وعدل البيانات
```

#### 2. **إعداد التطبيق (Flutter)**
```bash
# 1. استنساخ المشروع
git clone <repository-url>
cd new

# 2. تثبيت التبعيات
flutter pub get

# 3. تعديل إعدادات API
# افتح lib/config/config.dart وعدل عنوان الخادم
```

#### 3. **تشغيل التطبيق**
```bash
# تشغيل التطبيق
flutter run

# أو بناء التطبيق للإنتاج
flutter build apk
flutter build ios
```
